Calculate Steinhart-Hart Equation coefficients from three meaurments:
low temperature, mid-range temperature and high temperature.
These can be taken from the manufacturers measurement

eg for Thermistor NCP15XH193F03RC off data specs 0C = 27218.6 0ohms, 20C = 12080.5 0ohms, 40C = 5833.6 ohms
TUTORIAL instructions: With prefilled fields, press button "Calculate A,B,C"
followed by button CaluclateTemperature for a test 10000 ohms- the Result should be close to 25C
